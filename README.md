# Compliance & Regulatory Hub

Enterprise HTML documentation for U.S. financial crime compliance programs — covering KYC, AML, BSA, CDD/EDD, OFAC sanctions screening, regulatory reporting, and audit governance.

## Overview

This site provides internal reference documentation for regulated financial institutions operating under U.S. Bank Secrecy Act (BSA), FinCEN, and OFAC requirements. Content is structured around the core compliance program pillars and operational reference needs of compliance analysts, BSA officers, and operations staff.

Static HTML site hosted via GitHub Pages. No build tools or backend required.

## Pages

| Page | Topic |
|------|-------|
| `index.html` | Overview & Regulatory Framework |
| `kyc.html` | Know Your Customer (KYC) — CIP, identity verification, risk scoring |
| `aml.html` | Anti-Money Laundering — program structure, typologies, TMS rules |
| `bsa.html` | Bank Secrecy Act — reporting thresholds, CTR/SAR/FBAR, recordkeeping |
| `cdd.html` | Customer Due Diligence — CDD/EDD tiers, beneficial ownership |
| `sanctions.html` | OFAC Sanctions Screening — SDN list, blocking/rejecting, reporting |
| `reporting.html` | Regulatory Reporting — SAR standards, CTR procedures, FinCEN timelines |
| `audit.html` | Audit Trail & Compliance Logging — recordkeeping, audit standards, exam prep |
